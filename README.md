# bhwi - Blogger Heaven Widget Instagram

# Example

```
git clone git@github.com:cyrilkyburz/bhwi.git
cd bhwi/example
Open index.html in a browser
```

# Setup

```
git clone git@github.com:cyrilkyburz/bhwi.git
cd bhwi
npm install bower gulp tsd -g
npm install
tsd reinstall --save --overwrite
```

# Requirements 

* nvm

# Gulp

## Development

(Auto compiling)

```
gulp 
```

## Compile

```
gulp compile
```

## Release

```
gulp compile --type production
```
