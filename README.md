# bhwi - Blogger Heaven Widget Instagram

# Setup

```
git clone git@git.renuo.ch:cyrilkyburz/bhwi.git
cd bhwi
npm install bower gulp tsd -g
tsd reinstall --save --overwrite
```


# Requirements 

* iojs (or Node)
* npm

# Gulp

## Development

(Auto compiling)

```
gulp 
```

## Compile

```
gulp compile
```

## Release

```
gulp release
```
